Documentation

myFlix is a web application designed for users to search for movies. Users can register and create a profile in which they can create lists to save their favorite movies based on title, genres, directors, actors, etc.

Allows new users to register /users POST A JSON object containing data about the new registered user:

{
Username : "firstuser",
Password : "test1234",
Email : "test@gmail.com",
Birthday : "12.29.1986"
}

JSON object containing data about the new user info:

{

} }
Allow users to update info /users/[Username] PUT JSON object containing data about current username:

{
Username : "firstuser",
Password : "test1234",
Email : "test@gmail.com",
Birthday : "12.29.1986"
}

Allows users to add movies to their favorites /users/[Username]/ favorites/[FavMovie] POST JSON object containing the user's favorite movies

{
Favorites : {
FavMovie : "Grandm's Boy",
Director : "Nicholaus Goossenn" }
}
Allow users to remove movies from their favorites /users/[Username]/ favorites/[FavMovie] DELETE None Response message indicating whether the movie was successfully removed from the user's favorites
Allows existing users to delete their profile /users/[Username]/ DELETE None A text message indicating whether the user's account was succesfully removed.
